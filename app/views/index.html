Form examples

Some examples of common kinds of form, using components and patterns from the GOV.UK Design System.

Example 1: A simple form

Features:

  • There's one question per page
  • Each page has space for help text
  • There's no skip logic or sections
  • The pet species question has a free text 'other' option
  • The pet description question has a character limit
  • The check your answers page includes a declaration
