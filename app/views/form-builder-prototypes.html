Admin prototype

View the admin prototype

A form builder with the following features:

  • the interface and forms are accessible and mobile friendly
  • there's a task list to take you through the steps of creating a form
  • you create one question at a time, using a looping wizard
  • questions are designed via a form, rather than directly edited
  • questions can be repeated within the form runner
  • questions can be grouped and repeated within the form runner
Why did we build this prototype?

This prototype is for testing an iteration of ‘add another’ feature where users can create repeating sets of questions through the route journey.
